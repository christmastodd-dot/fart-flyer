# Fart Flyer — Milestone 3: iOS Build

## Play in browser right now
Open `web/index.html` in any browser. No server needed.

---

## Build for iOS

### Prerequisites
| Tool | Version | Notes |
|---|---|---|
| Node.js | 18 + | |
| Xcode | 15 + | Mac only |
| Apple Developer account | — | Free account works for device testing; paid ($99/yr) needed for App Store |

### Steps

```bash
# 1. Install Capacitor and dependencies
cd milestone3
npm install

# 2. Generate the iOS project (first time only)
npx cap add ios

# 3. Copy web files into the native project
npx cap sync

# 4. Open in Xcode — sign, select device, and run
npx cap open ios
```

In Xcode:
1. Select your **Team** under *Signing & Capabilities*
2. Pick your connected iPhone as the run target
3. Press ▶ Run

### App icons

Capacitor expects icons in `ios/App/App/Assets.xcassets/AppIcon.appiconset/`.
The easiest way to generate all required sizes from a single 1024×1024 source:

```bash
npm install -g @capacitor/assets
npx capacitor-assets generate --ios
```

You'll need to provide `assets/icon.png` (1024×1024 px, no transparency) and
`assets/splash.png` (2732×2732 px). A simple pixel-art yellow girl on a sky-blue
background works perfectly.

### Haptic feedback

`haptics.js` calls `window.Capacitor.Plugins.Haptics` when running inside the
native shell, and falls back to `navigator.vibrate` in the browser.
Haptics require a real device — the iOS Simulator does not vibrate.

### Subsequent updates

After changing any file in `web/`:
```bash
npx cap sync   # copies updated web/ into the native project
```
Then re-run from Xcode.

## Difficulty levels

| Level | Score to reach | Pipe gap | Speed bonus |
|---|---|---|---|
| 1 | 0  | 120 px | — |
| 2 | 5  | 114 px | +0.5 |
| 3 | 10 | 108 px | +1.1 |
| 4 | 20 | 100 px | +1.8 |
| 5 | 35 |  92 px | +2.6 |

A full-screen **LEVEL UP!** overlay appears for 1.5 seconds (tap to dismiss early)
whenever the player crosses a threshold.

## File structure

```
milestone3/
  web/                   ← the entire game (opened by Capacitor's WKWebView)
    index.html           ← viewport-fit=cover, apple-mobile-web-app-capable
    sprites.js           ← pixel art data + drawSprite()
    audio.js             ← procedural Web Audio sounds
    haptics.js           ← Capacitor haptics + navigator.vibrate fallback
    renderer.js          ← all canvas draw calls
    game.js              ← state machine, physics, levels, input, loop
  capacitor.config.json  ← app ID, webDir, plugin config
  package.json           ← @capacitor/core, @capacitor/ios, @capacitor/haptics
  ios/                   ← generated by `npx cap add ios` (not committed)
```
